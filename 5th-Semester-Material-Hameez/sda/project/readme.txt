1. unzip the project
2. open visual studio code and select that specific folder
3. open windows power shell, cmd, or linux terminal
4. type pip install Flask
and then flask will be installed in your pc.
5. open XAMP
6. start mysql and apache
7. click on import and upload the foodpanda.sql file there and press snter.
Db will be setup now.
8. Now run the main.py file and wait for it to generate local host's ip.
9. open the local host's up and your project will be started.
